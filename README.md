# BMI Calculator

A simple BMI (Body Mass Index) calculator built with Python.

## Overview

This project was created to practice Python fundamentals such as:

- Functions
- Modules
- User input handling
- Exception handling
- Conditional statements
- Code organization

The program asks the user for weight and height, validates the input, calculates the BMI, and displays the health classification.

---

## Features

✔ User input for weight and height  
✔ Input validation using `try/except`  
✔ BMI calculation  
✔ BMI classification system  
✔ Data confirmation before calculation  
✔ Organized code using separate modules
